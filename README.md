### FreeAI Hub: Models

## What is this project?

FreeAI Hub is a project inspired by Hugging Face, directly adapted for artificial intelligence. This project allows you to:
- Use existing models (GPT-2 or others).

- Test them (they help you with your own AI on Telegram or Discord).

- Train models based on these.

In addition, it can help with:

- Text to images.

- Images to text.

- More.
